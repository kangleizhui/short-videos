# 异常处理指南

## API 解析失败

| 现象 | 原因 | 处理 |
|------|------|------|
| HTTP 500 | 抖音 WAF 拦截 / 服务异常 | 重试1次，仍失败则告知用户「解析服务暂不可用，稍后再试」 |
| HTTP 403 | 身份码无效/已停用 | 检查身份码是否正确传入 |
| HTTP 429 | 每日/总次数用完 | 引导购买套餐续费 |
| 超时（>30s无响应） | 抖音图文笔记解析慢 | 重试1次，设置 60s 超时 |

## 素材下载失败

| 现象 | 原因 | 处理 |
|------|------|------|
| 视频URL下载返回403 | CDN 防盗链/URL过期 | 直接用 `data.url` |
| 动图段下载失败 | 抖音 CDN geo-block / 过期 | 跳过这段，只发成功下载的动图段 + 静图 + 配乐 |
| 图片下载失败 | 代理临时故障 | 重试1次，跳过该张图片 |
| 配乐下载失败 | CDN 问题 | 跳过配乐，只发视频段/图片 |
